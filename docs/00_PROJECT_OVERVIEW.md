# EKOSISTEM DIGITAL PESANTREN

## Tujuan

Membangun sistem digital terintegrasi untuk operasional pesantren.

Sistem dirancang secara modular agar dapat berkembang bertahap dari administrasi dasar menuju ekosistem digital penuh.

---

## Tech Stack

Frontend:
- React
- Axios

Backend:
- Node.js
- Express

Database:
- PostgreSQL

---

## Modul Sistem

1. Pendidikan
2. Keamanan
3. Keuangan
4. RFID
5. Kantin Digital
6. Wali Santri
7. Administrasi

---

## Prinsip Pengembangan

- Modular
- Realtime
- Offline First
- Mudah Digunakan Operator
- Semua transaksi terpusat
- Siap integrasi RFID
